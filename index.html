Security & Privacy

All file processing happens in your browser. No data ever leaves your device.

File handling

Files you upload are parsed in-memory by JavaScript libraries. They are never sent to any server.

Local AI (offline)

The local AI (Qwen2.5-1.5B via WebLLM) runs entirely in your browser via WebGPU. Nothing is sent anywhere. The model file is downloaded once (~1GB) and cached for offline reuse. Warning: can cause significant lag on low-end devices.

Contact & Feedback forms

When you submit the contact or feedback form, your message is sent via FormSubmit.co to Newo's email (Owennelfrey@gmail.com). Only the form fields you fill in are transmitted. No file data is ever sent.

XSS protection

All cell content is HTML-escaped. AI responses go through a custom HTML sanitizer that strips javascript: URLs, on* event handlers, and dangerous tags.

Limits

Max file size: 50 MB. Max chat history: 30 messages. Max table render: 1000 rows per page. Max PDF pages: 50.

What we don't do

  • No analytics, telemetry, or tracking
  • No cookies (only localStorage for preferences)
  • No service workers or background sync
  • No hidden backend — the entire app is one HTML file
